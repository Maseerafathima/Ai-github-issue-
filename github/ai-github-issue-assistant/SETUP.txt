## To Push to GitHub

### 1. Install Git (if not already installed)
- Download from: https://git-scm.com/download/win
- Run installer with default options
- Restart your terminal after installation

### 2. Configure Git
```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@github.com"
```

### 3. Create Repository on GitHub
- Go to: https://github.com/new
- Repository name: `ai-github-issue-assistant`
- Choose: Public
- Click: Create Repository
- Copy the HTTPS URL

### 4. Push from Command Prompt
```bash
cd c:\Users\gousi\OneDrive\Desktop\github\ai-github-issue-assistant

git init
git remote add origin https://github.com/YOUR_USERNAME/ai-github-issue-assistant.git
git branch -M main
git add .
git commit -m "Initial commit: AI GitHub Issue Assistant"
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username

### 5. Verify
- Visit: https://github.com/YOUR_USERNAME/ai-github-issue-assistant
- All files should be visible

## Troubleshooting

### App won't start
- Ensure Python 3.9+ is installed
- Check: `python --version`
- Reinstall dependencies: `pip install -r requirements.txt`

### "GEMINI_API_KEY not found"
- Edit `.env` file
- Add your API key: `GEMINI_API_KEY=your_key_here`
- Restart the app

### "Invalid GitHub URL"
- Use format: `https://github.com/owner/repo`
- Repository must be public
- Remove trailing slashes

### Git not found
- Install Git from: https://git-scm.com/download/win
- Restart PowerShell/Command Prompt after installation
